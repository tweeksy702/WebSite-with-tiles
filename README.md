# Теплая Плитка (Портфолио)

Статический сайт на HTML/CSS/JS.

## Локальный запуск

1. Откройте терминал в папке проекта.
2. Запустите сервер:

```bash
python -m http.server 5500
```

3. Откройте в браузере:

`http://localhost:5500/`

## Публикация (рекомендуется: Netlify)

Самый простой способ получить публичную ссылку:

1. Перейдите на [https://app.netlify.com/drop](https://app.netlify.com/drop).
2. Перетащите в окно всю папку проекта.
3. Через несколько секунд получите постоянную публичную ссылку вида `https://имя-сайта.netlify.app`.

### Обновление сайта

После изменений в проекте повторно перетащите папку на Netlify Drop, либо подключите GitHub-репозиторий для автодеплоя.

## Альтернатива: GitHub Pages

1. Создайте репозиторий на GitHub и загрузите файлы проекта.
2. В настройках репозитория откройте раздел `Pages`.
3. В `Build and deployment` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` и папку `/ (root)`
4. Сохраните и дождитесь публикации.

Сайт будет доступен по ссылке вида:
`https://<username>.github.io/<repo>/`
